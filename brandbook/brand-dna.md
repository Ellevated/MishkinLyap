## Phase 0: Convergence Shield

### Emotive Narrative

> Когда кто-то открывает "Мишкин Ляп", он должен почувствовать себя как на полке с плюшевыми игрушками в уютном зоомагазине — тёплом, мягком, где хочется остаться. Мишка — добрый ведущий, который "ляпает" зверят сверху. Эстетика ближе к Pusheen, чем к Angry Birds. Если бы этот бренд был физическим пространством, это был бы маленький зоомагазин с мягким светом, деревянными полками и запахом печенья. Вайб — милота и тепло, никогда агрессивно или крикливо.

### Chosen Direction: C — Experimental

**"Советская детская иллюстрация"**

| Параметр | Значение |
|----------|---------|
| Mood | винтажный, рукодельный, текстурный, ностальгический, тактильный |
| Color Temperature | Warm muted |
| Typography Feel | Hand-drawn, ретро-стиль |
| Reference World | Советская детская иллюстрация 1970-х (Сутеев, Чарушин) |

**Почему:** Ностальгия для основной аудитории (женщины 55+, выросшие на этих книжках). Уникально на платформе — ни одна drop-merge игра не использует этот стиль. Тёплые muted тона идеально подходят для casual gaming.

### Anti-Convergence Banned Defaults

| Banned | Вместо |
|--------|--------|
| Inter, system-ui | Рукодельные / ретро шрифты |
| Purple/blue gradients | Тёплые охра, терракота, горчица |
| Rounded white cards on gray | Текстурные поверхности, "бумага" |
| "Modern minimalist" | "Рукодельный, книжный, тёплый" |
| Stock photo style | Иллюстративный, как из детской книжки |

## Phase 2: Brand DNA

### Archetype: Innocent (Невинный)

"Мишкин Ляп" — про простое удовольствие, отсутствие стресса, тепло и честность. Аудитория 55+ хочет спокойствия, не вызовов. Innocent = "жизнь может быть простой и доброй". Идеально совпадает с советской детской книжкой — там та же философия добра и мягкости.

### Visual Descriptors (Style Seed)

`тёплый, рукодельный, текстурный, ностальгический, мягкий, иллюстративный, округлый`

### Brand Personality Spectrum

```
Formal  --------[--]- Casual         → 4/5 (casual, дружелюбно)

Traditional -[-------] Modern        → 2/5 (ретро-ностальгия)
Complex --------[--]- Simple         → 5/5 (предельно просто)
```

## Phase 3: Voice Framework

### Voice Definition

**"Мы звучим тепло, просто и по-доброму — но никогда снисходительно."**

### Vocabulary Rules

| Используем | Не используем | Почему |
|-----------|--------------|--------|
| зверята, дружок | юниты, объекты | Тёплый язык vs техничный |
| собрать, соединить | мёрджить, комбинировать | Простые русские глаголы |
| играй | жми, кликай | Мягкий призыв, не директива |
| отлично! молодец! | GG, респект, gg wp | Аудитория 55+ не знает геймерский сленг |
| попробуй ещё | ты проиграл | Innocent не наказывает, а подбадривает |
| зверёк | моб, спрайт, entity | Внутри мира игры, не снаружи |
| подсказка | туториал, гайд | Русский, понятный |

### Tone by Context

| Контекст | Тон | Пример копирайта |
|---------|-----|-----------------|
| Каталог Яндекс Игр | Тёплый, приглашающий | "Милые зверята ждут тебя! Собери всех — от хомячка до мишки" |
| Game Over | Ободряющий, мягкий | "Отличная игра! Попробуешь ещё разок?" |
| Rewarded Video | Дружелюбное предложение | "Посмотри видео — зверята дадут ещё попытку!" |
| Push / Напоминание | Тёплое приглашение | "Мишка скучает! Загляни на минутку?" |
| Описание в магазине | Простой, чёткий | "Роняй зверят, соединяй одинаковых — собери мишку!" |
| Ошибка | Не виним пользователя | "Ой, что-то пошло не так. Попробуй ещё раз!" |

### Copywriting System Prompt

```
You are the brand voice for Мишкин Ляп (Mishkin Lyap) — a cozy drop-merge game with cute animals.

Tone: Warm, simple, and kind — but never condescending.
Archetype: Innocent — life can be simple and good.
Audience: Women 35-65, casual gamers on Yandex Games. Low tech literacy, high emotional intelligence.

Always use: зверята, собрать, соединить, играй, отлично, попробуй, дружок
Never use: юниты, мёрджить, кликай, ты проиграл, GG, туториал, респект

Style: Soviet children's book illustration era — warm, nostalgic, handcrafted feel.
Write in Russian. Short sentences. No jargon. Imagine you're writing a postcard to a kind grandmother.
```
