## ALWAYS Use

- Brand tokens from `brandbook/brand-tokens.css` (import via `@import`)
- OKLCH color notation in new CSS
- Font pair: Marmelad (display) + Nunito (body)
- Semantic color tokens (`--color-cta`, `--color-bg`) not raw HEX values
- `prefers-reduced-motion` media query for ALL animations
- `prefers-color-scheme: dark` for dark mode
- Min 48x48px touch targets (55+ audience)
- Min 18px body text, 14px absolute minimum
- Dark text (#3D2B1F) on Ochre CTA — NOT white text

## NEVER Use

- Inter, system-ui, or Roboto as primary fonts
- Raw HEX values instead of CSS custom properties
- Colors not in the approved palette above
- White text on Ochre (#D4A24C) backgrounds — fails WCAG
- Animations longer than 500ms (except logo intro)
- Purple/blue gradients or rounded-white-card aesthetic
- Text smaller than 14px anywhere
- Touch targets smaller than 48x48px
